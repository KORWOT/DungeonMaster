═══════════════════════════════════════════════
📋 이번 주 필수 작업 (1시간 30분)
═══════════════════════════════════════════════

✅ 1. 파이어볼 스킬 생성 (30분)
   - Project 우클릭 → Create → Game → Skill Data
   - 파일명: Fireball_Skill
   - Inspector에서 데미지 50, 쿨다운 3초 설정

✅ 2. SkillManager 설정 (15분)
   - Resources 폴더에 SkillManager.asset 생성
   - All Skills에 Fireball_Skill 할당

✅ 3. 테스트 환경 구성 (20분)
   - Hierarchy에 SkillTestManager 오브젝트 생성
   - SkillPerformanceTest 컴포넌트 추가

✅ 4. 동작 테스트 (10분)
   - Play 버튼 → Inspector 우클릭 → 성능 테스트 실행
   - Console에서 결과 확인

✅ 5. 추가 스킬 생성 (45분)
   - 힐링 스킬 (Type: Heal, FixedValue: 30)
   - 공격 버프 (Type: AttackBuff, PercentValue: 20)
   - 디버프 스킬 (Type: SpeedDebuff, PercentValue: -30)

═══════════════════════════════════════════════
🗺️ 전체 로드맵 (10-14주)
═══════════════════════════════════════════════

📅 Phase 1 (1-2주): 스킬 시스템 완성
   → 다양한 스킬 타입, 스케일링, 몬스터 연동

📅 Phase 2 (2-3주): UI 시스템
   → 스킬 설명창, 레벨업 미리보기, 몬스터 관리 UI

📅 Phase 3 (3-4주): 던전 시스템
   → 맵, 배치, 웨이브, 실시간 전투

📅 Phase 4 (1-2주): 성능 최적화 ⚡ 캐싱 시점!
   → 배치 기반 캐싱, Object Pooling, 렌더링 최적화

📅 Phase 5 (2-3주): 최종 완성
   → 밸런싱, 사운드, 이펙트, 모바일 최적화

═══════════════════════════════════════════════
📊 성능 목표
═══════════════════════════════════════════════

🎯 현재 목표: 초당 66회 스킬 처리 (200마리 × 3초마다)
📱 모바일 최적화: 메모리 40KB, 처리시간 2.3ms/초
⚡ 캐싱 효과: 히트율 95%+, 10-50배 빠른 응답
